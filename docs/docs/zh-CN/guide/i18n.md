# 国际化

## 配置

同 elememt-plus 一样可以通过 ElConfigProvider 注入国际化配置

::: demo
@/demo/I18n/base.vue
:::

## 可配置字段

```js
const locale = {
  pro: {
    form: {
      submit: '提交',
      reset: '重置',
    },
    crud: {
      add: '新增',
      edit: '编辑',
      detail: '查看',
      del: '删除',
      submit: '提交',
      reset: '重置',
      search: '搜索',
      searchReset: '重置',
    },
  },
}
```
