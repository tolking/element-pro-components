# Internationalization

## configuration

Like elememt-plus, international configuration can be inject through ElConfigProvider

::: demo
@/demo/I18n/base.vue
:::

## Configurable fields

```js
const locale = {
  pro: {
    form: {
      submit: 'Submit',
      reset: 'Reset',
      prev: 'Prev',
      next: 'Next',
    },
    search: {
      submit: 'Search',
      reset: 'Reset',
      prev: 'Prev',
      next: 'Next',
    },
    crud: {
      add: 'Add',
      edit: 'Edit',
      detail: 'Detail',
      del: 'Delete',
      submit: 'Submit',
      reset: 'Reset',
      prev: 'Prev',
      next: 'Next',
      search: 'Search',
      searchReset: 'Reset',
      searchPrev: 'Prev',
      searchNext: 'Next',
    },
  },
}
```
